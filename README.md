# uac-system-capacity-analytics.o
System Capacity & Care Load Analytics for Unaccompanied Children

This project analyzes daily capacity and care load across the CBP–HHS
Unaccompanied Alien Children (UAC) program.

## Objective
- Monitor total system care load
- Identify capacity strain and backlog periods
- Support data-driven healthcare and shelter planning

## Data
Source: U.S. Department of Health and Human Services  
Period: 2023–2025

## Tools
- Python
- Pandas
- Streamlit
- VS Code

## How to Run
pip install -r requirements.txt  
cd UAC_Capacity_Analytics;streamlit run app/streamlit_app.py
